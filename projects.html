Goal/ purpose of the project:implement a file compression tool using the Huffman algorithm. The project aims to reduce the storage size of files without losing any information
Details:The project involves creating two programs: compress and uncompress. The compress program takes an arbitrary file as input, creates a Huffman code for the contents of the file, and uses the Huffman code to encode the file. The compressed version of the file is then written to a new file with a header to allow for uncompression. The uncompress program takes a compressed file created by the compress program as input, reads the header of the file to construct a Huffman code, uses the Huffman code to decode the compressed file, and writes the uncompressed version of the file to a new file.
